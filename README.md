# discord-patcher
патчер дискорда для Windows на основе запрета, позволяет заходить в дискорд с РФ без использования впн или других сервисов.
# Как использовать?
1. Открываем архив "discord patcher.rar"
2. Перекидываем папку с тем же названием куда угодно, главное не в другую папку.
3. Открываем discord patcher.bat
4. после того как у вас написались странные символы нажимаем Enter
Если что то не понятно то есть видео: https://youtu.be/LvvarWlwhSM
# Решение частых ошибок
Для начала попробуйте отключить брандмауэр
Если антивирус жалуется на файл то отключите его, он реагирует на WinDivert, этот инструмент необходим для работы приложения и он не является вирусом.
